# Active directory components 

Physical and logical.

physical components:
domain controller - hosts a copy of AD DS directory store, allows admin access and stuff, very important
AD DS data store - contains all database store,
contains the NTDS.dit file.

Logical components:

AD DS schema - like a blueprint/rulebook for object creation

Domain - used to group and manage objects , like a boundry

Trees - hierarchy of domains

Forest - collection of domain trees
