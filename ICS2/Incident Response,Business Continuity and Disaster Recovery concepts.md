
## incident terminology

breach - the loss of control,compromise,unauthorized disclosure,unauthorized acquisition or similar occurence.

Event - any observable occurence in a network

Exploit - A particular attack(exploits system vulnerability)

Incident - An event that actually or potentially jeopardizes the confidentiality,integrity, or availability.

Intrusion - delibrate security incident in which an intruder gains, or attempts to gain, access to a system or resource

Threat - circumstance or event with the potential to adversely impack organizational operations

Vulnerability - weakness in a system.

Zero day - Previously unknown system vulnerability.
