
subject - anything that requests access to a service/system/file etc.

object - anything that the subject tries to access.

rules - defines who can access what objects and when.

physical control - locks,physical access
logical control - passwords,biometric etc.
administrative control - laws/policies

controls help reduce risk to a tolerable level

user provisioning - creating,disabling,modifying accounts

permission/privilage creep - having unneccasary privs from previous access.
